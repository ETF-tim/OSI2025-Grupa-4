# Projekat iz Osnova softverskog inženjerstva 
Ovo je repozitorijum namjenjen za izvorni kod studentskog projekta iz Osnova softverskog inženjerstva na Elektrotehničkom fakultetu Univerziteta u Banjoj Luci.

Struktura projekta: 

> /src - Izvršni kod
> /include - Header fajlovi
> /docs - Dokumentacija vezana za projekat
> /tests - Testovi
> /data - Podaci
